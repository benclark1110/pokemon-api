# Pokemon API

## Tech Stack:
- Python
- MongoDB Atlas

## HOW TO:
1. 'pipenv install' to gather dependencies
2. 'pipenv shell' to start the virtual environment
3. 'python3 insert_pokemon.py' to load pokemon
4. 'flask run' to start api
